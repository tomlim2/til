SQL vs noSQL

## SQL(Structured Query Language)

관계형 데이터베이스
장점: 데이터 관리가 쉽다.
단점: 수정이 힘들다.

ex) mySQL, SQLite

## noSQL

비관계형 데이터베이스의 통합.
장점: 수정이 쉽다.
단점: 중복 데이터 수정이 힘들다.
